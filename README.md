# Job Portal

A static multi-page job portal website built with HTML and CSS.

## Live Site

- https://rameshwaryar77-lgtm.github.io/Job-Portal/

## Repository

- https://github.com/rameshwaryar77-lgtm/Job-Portal

## Pages Included

- `index.html` - homepage and landing section
- `about.html` - overview of the platform and role expectations
- `job.html` - featured job listings
- `contact.html` - developer and contact section
- `login.html` - login page
- `form.html` - job application form

## Project Structure

- `styles.css` - shared responsive styles for the main public pages
- `*.html` - static site pages
- `*.jpg`, `*.jpeg`, `video.mp4` - local media assets

## Improvements Added

- moved the site to a single `index.html` homepage
- added a shared stylesheet for easier maintenance
- improved navigation consistency across main pages
- made the homepage, about, jobs, and contact pages responsive
- added a README for GitHub visitors

## Run Locally

Open `index.html` in a browser.

## Deployment

This project is deployed with GitHub Pages from the `main` branch root.
